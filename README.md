# Data Annotation Practice Exam

## Overview

A comprehensive practice exam application built with React and Next.js designed to help candidates prepare for Data Annotation assessments. The exam consists of 47 questions covering various technical and analytical skills required for data annotation and AI training roles.

## Description

This interactive web application provides a full-featured exam experience with:

- **47 diverse questions** spanning multiple categories including:
  - Data Structures and Algorithms
  - Programming (Python, JavaScript, Java)
  - SQL and Database concepts
  - Machine Learning and Data Science
  - Content Moderation and Ethics
  - Reading Comprehension
  - Instruction Following

- **8 different question types**:
  - Multiple Choice
  - Instruction Following
  - Reading Comprehension
  - Named Entity Recognition (NER) Tagging
  - AI Response Evaluation
  - Creative Writing
  - Coding Challenges
  - Short Answer

## Features

- Real-time progress tracking
- Question navigation (previous/next)
- Answer counter showing completed questions
- Comprehensive results review with explanations
- Expected answers and key points for each question
- Retake functionality
- Responsive design with clean UI

## Usage

The application allows users to:
1. Navigate through questions sequentially
2. Review and modify answers before submission
3. Submit the completed exam
4. Review all answers with detailed explanations
5. Retake the exam for practice

## Important Note

This is a practice exam intended for self-assessment and learning. Users should complete the exam independently without AI assistance for accurate skill evaluation. After completing the exam once, users are encouraged to use AI tools to learn and improve on topics where they need development. All questions originate from general personal work experience without delving deep into any real projects.
